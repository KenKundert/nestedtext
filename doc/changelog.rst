****************
Language changes
****************

.. currentmodule:: nestedtext

Currently the language and the Python implementation share version numbers.  
Since the language is more stable than the implementation, you will see versions 
that include no changes to the language.

Latest development version
--------------------------

    | Version: 1.3.1
    | Released: 2021-03-03

    - Deprecate quoted dictionary keys.
    - Add multi-line dictionary keys to replace quoted keys.
    - Add single-line lists and dictionaries.

    .. warning::

        Be aware that this version is not backward compatible because it no 
        longer supports quoted dictionary keys.
